# C++ 语法面试题整理

## 1. 自增自减运算符 (++/--)

### 前置与后置的区别

```cpp
int a = 5;
int b = ++a;  // 前置：先自增，再返回。a=6, b=6
int c = a++;  // 后置：先返回，再自增。c=6, a=7
```

**关键点：**
- `++a`：返回自增后的值（左值）
- `a++`：返回自增前的值（右值）

### 常见面试陷阱

#### 陷阱 1：表达式求值顺序

```cpp
int i = 1;
int result = i++ + ++i + i++;
// 结果是多少？
```

**易错点：** C++ 中同一表达式内多次修改同一变量是**未定义行为**（UB）。

**正确答案：** 这段代码的行为是未定义的，不同编译器可能给出不同结果。

#### 陷阱 2：函数参数求值顺序

```cpp
int i = 0;
printf("%d %d %d\n", i++, i++, i++);
```

**易错点：** 函数参数的求值顺序在 C++17 之前是未指定的。

### 实现原理

```cpp
// 前置++的实现
T& operator++() {
    *this += 1;
    return *this;  // 返回引用
}

// 后置++的实现
T operator++(int) {  // int 是占位符，用于区分
    T old = *this;
    ++(*this);       // 调用前置版本
    return old;      // 返回副本
}
```

### 最佳实践

1. **避免在同一表达式中多次修改同一变量**
2. **前置自增通常比后置自增效率更高**（尤其对于自定义类型）
3. **优先使用前置形式** `++i`，除非明确需要旧值
